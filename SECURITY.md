# Security policy

## Reporting a vulnerability

Email: smigolsmigol@protonmail.com

Acknowledgment SLA: 48 hours.
Critical-fix SLA: 7 days.

## Supported versions

| Version | Supported |
|---------|-----------|
| latest  | Yes       |

## Default for the f3d1 ecosystem

This file is the org-level default. Every repo under `smigolsmigol/*` that does not ship its own `SECURITY.md` inherits this policy automatically.

For runtime-specific concerns on a particular package, the relevant repo's `SECURITY.md` (when present) overrides this file.

## Scope

In scope: any of the actively-maintained smigolsmigol/* repos (f3dx, llmkit, f3dx-cache, pydantic-cal, f3dx-router, f3dx-bench, tracewright, keyguard, .github).

Out of scope: forks, archived repos, third-party dependencies (file with the upstream).
